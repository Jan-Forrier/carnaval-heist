# Heist viert Carnaval

Een Next.js website voor het carnaval in Heist, gebouwd met TypeScript en Tailwind CSS.

## Getting Started

Installeer de dependencies:

```bash
npm install
```

Run de development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in je browser om de site te bekijken.

## Build

Om een productie build te maken:

```bash
npm run build
npm start
```
